= Apagar uma nuvem
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#EXCLUIR# [.api-doc-code-block]#`/accounts/{account_id}/topology/v1/clouds/{cloud_id}`#

Exclui a instância de nuvem solicitada e quaisquer clusters e volumes não gerenciados associados a essa nuvem específica.



== Parâmetros

[cols="5*"]
|===
| Nome | Tipo | Em | Obrigatório | Descrição 


| id_da_conta | string | caminho | Verdadeiro  a| 
ID do recurso de conta que o contém

* exemplo: {{.Account}}




| id_da_nuvem | string | caminho | Verdadeiro  a| 
ID da coleção de nuvens a ser listada

|===


== Resposta

[source]
----
Status: 204, Indicates if the cloud resource was deleted.
