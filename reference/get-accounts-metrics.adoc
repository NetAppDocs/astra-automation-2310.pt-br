= Recuperar todas as métricas de uma conta
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/accounts/{account_id}/metrics`#

Retorna todas as métricas da conta



== Parâmetros

[cols="5*"]
|===
| Nome | Tipo | Em | Obrigatório | Descrição 


| id_da_conta | string | caminho | Verdadeiro  a| 
ID do recurso de conta que o contém

* exemplo: {{.Account}}


|===


== Resposta

[source]
----
Status: 200, Returns a OpenMetrics text document containing all metrics for the account.
----
