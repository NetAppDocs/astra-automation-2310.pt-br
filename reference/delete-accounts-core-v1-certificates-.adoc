= Excluir um certificado
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#EXCLUIR# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/certificates/{certificate_id}`#

Remove o recurso de API de Certificado especificado.



== Parâmetros

[cols="5*"]
|===
| Nome | Tipo | Em | Obrigatório | Descrição 


| id_da_conta | string | caminho | Verdadeiro  a| 
ID do recurso de conta que o contém

* exemplo: {{.Account}}




| certificate_id | string | caminho | Verdadeiro  a| 
ID da coleção de certificados a ser listada

|===


== Resposta

[source]
----
Status: 204, Indicates if the certificate resource was deleted.
----
