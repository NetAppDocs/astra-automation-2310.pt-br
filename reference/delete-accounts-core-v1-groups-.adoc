= Excluir um grupo
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#EXCLUIR# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/groups/{group_id}`#

Remove o recurso da API de grupo especificado.

*Efeitos Colaterais*

* Os seguintes recursos associados ao grupo foram excluídos:
* Vínculos de papéis referentes ao grupo




== Parâmetros

[cols="5*"]
|===
| Nome | Tipo | Em | Obrigatório | Descrição 


| id_da_conta | string | caminho | Verdadeiro  a| 
ID do recurso de conta que o contém

* exemplo: {{.Account}}




| id_do_grupo | string | caminho | Verdadeiro  a| 
ID da coleção de grupos a ser listada

|===


== Resposta

[source]
----
Status: 204, Indicates if the group resource was deleted.
----
