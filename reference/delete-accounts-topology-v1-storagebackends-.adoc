= Excluir um backend de armazenamento
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#EXCLUIR# [.api-doc-code-block]#`/accounts/{account_id}/topology/v1/storageBackends/{storageBackend_id}`#

Remove o recurso da API de backend de armazenamento especificado.

*Efeitos Colaterais*

* Todos os recursos backendConfig associados ao backend de armazenamento são removidos.
* Todos os recursos backendNode associados ao backend de armazenamento são removidos.
* O sistema de armazenamento pode ser removido.




== Parâmetros

[cols="5*"]
|===
| Nome | Tipo | Em | Obrigatório | Descrição 


| id_da_conta | string | caminho | Verdadeiro  a| 
ID do recurso de conta que o contém

* exemplo: {{.Account}}




| storageBackend_id | string | caminho | Verdadeiro  a| 
ID da coleção storageBackend a ser listada

|===


== Resposta

[source]
----
Status: 204, Indicates if the storageBackend resource was deleted.
