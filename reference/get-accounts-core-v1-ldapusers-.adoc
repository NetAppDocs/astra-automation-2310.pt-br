= Recuperar um usuário LDAP
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/ldapUsers/{ldapUser_id}`#

Retorna um objeto JSON que representa o recurso LDAP armazenado.



== Parâmetros

[cols="5*"]
|===
| Nome | Tipo | Em | Obrigatório | Descrição 


| id_da_conta | string | caminho | Verdadeiro  a| 
ID do recurso de conta que o contém

* exemplo: {{.Account}}




| ldapUser_id | string | caminho | Verdadeiro  a| 
ID da coleção ldapUser a ser listada

|===


== Resposta

[source]
----
Status: 200, Returns a JSON object representing the stored LDAP resource.
----
[cols="4*"]
|===
| Nome | Tipo | Obrigatório | Descrição 


| tipo | string | Verdadeiro  a| 
Tipo de mídia do recurso. Os valores definidos são:

* "aplicação/astra-ldapUser"




| versão | string | Verdadeiro  a| 
Versão do recurso. Os valores definidos são:

* "1,0"




| id | string | Verdadeiro  a| 
Identificador globalmente único do recurso. Os valores definidos são:

* Em conformidade com o esquema UUIDv4




| e-mail | string | Falso  a| 
String JSON representando o e-mail do usuário LDAP. Os valores definidos são:

* Comprimento mínimo: 3
* Comprimento máximo: 63




| firstName | string | Falso  a| 
String JSON representando o primeiro nome do usuário LDAP. Os valores definidos são:

* Comprimento mínimo: 1
* Comprimento máximo: 63




| lastName | string | Falso  a| 
String JSON representando o sobrenome do usuário LDAP. Os valores definidos são:

* Comprimento mínimo: 1
* Comprimento máximo: 63




| cn | string | Falso  a| 
String JSON representando o nome do grupo LDAP. Os valores definidos são:

* Comprimento mínimo: 1
* Comprimento máximo: 63




| dn | string | Falso  a| 
String JSON que representa o nome diferenciado do grupo LDAP. Os valores definidos são:

* Em conformidade com RFC 4514




| metadados | link:#type_astra_metadata["tipo_metadados_astra"] | Verdadeiro  a| 
Metadados especificados pelo cliente e pelo serviço associados ao recurso. Os valores definidos são:

* Conforme o esquema de metadados Astra
