= Excluir um Snapshot do aplicativo
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#EXCLUIR# [.api-doc-code-block]#`/accounts/{account_id}/k8s/v1/apps/{app_id}/appSnaps/{appSnap_id}`#

Remove o recurso Application Snapshot API especificado.

*Efeitos Colaterais*

* Se um Snapshot estiver em andamento, o Snapshot será cancelado.
* Se uma Snapshot for concluída, os dados da Snapshot serão excluídos.




== Parâmetros

[cols="5*"]
|===
| Nome | Tipo | Em | Obrigatório | Descrição 


| id_da_conta | string | caminho | Verdadeiro  a| 
ID do recurso de conta que o contém

* exemplo: {{.Account}}




| app_id | string | caminho | Verdadeiro  a| 
ID do recurso do aplicativo que o contém



| appSnap_id | string | caminho | Verdadeiro  a| 
ID da coleção appSnap a ser listada

|===


== Resposta

[source]
----
Status: 204, Indicates if the appSnap resource was deleted.
